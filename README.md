## Getting Started

First, run the development server:

1. install packeges:
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

2. run dev server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
